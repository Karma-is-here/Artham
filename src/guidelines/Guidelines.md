# Artham - Reflective Personal Finance App

## Overview
Artham is a reflective personal finance application that helps users understand and manage their finances through intelligent vault allocation.

## Design Philosophy
- Warm, modern, intelligent aesthetic
- Primary colors: Soft teal (#1C8B82), Gold (#F4B400), off-white backgrounds
- Fonts: Inter/Poppins
- Conversational, reflective, confidence-building tone

## Features
- Landing page with clear value proposition
- Comprehensive onboarding/settings flow
- Personalized dashboard with vault cards
- CSV upload for transaction analysis
- Risk appetite assessment
- Automatic vault calculations (Spend, Save, Grow, Protect)
- AI Financial Advisor Chatbot with conversational intelligence
